# Essayist

An AI-powered essay analysis tool designed for high school English students, particularly those preparing for exams like the Gaokao. Essayist uses advanced AI models to provide detailed scoring, feedback, and corrections on student essays based on content, language, and structure.

## Features

- **Intelligent Scoring**: Automatically scores essays on three dimensions: content (0-10), language (0-10), and structure (0-5), following professional grading standards.
- **Detailed Feedback**: Highlights good and bad parts of the essay with explanations.
- **AI Corrections**: Generates polished versions of essays with improvements in grammar, vocabulary, and flow.
- **Guided Writing Support**: Accepts specific writing requirements to tailor analysis.
- **Real-time Analysis**: Streaming results for immediate feedback.
- **Responsive UI**: Built with modern React components and animations.
