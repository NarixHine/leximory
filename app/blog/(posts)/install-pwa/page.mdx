# 安装 PWA 应用

> PWA（Progressive Web App）是一种渐进式网页应用，能够提供类似原生应用的体验。

Leximory 支持**将网页安装为本地应用**，获得更好的使用体验。本文将介绍如何在不同设备上安装 Leximory 的 PWA 版本。


## iOS 设备安装方法

![iOS PWA 安装](/install-pwa/ios.webp)

1. 使用 Safari 浏览器打开 Leximory
2. 点击右上角菜单
3. 选择**将此网站安装为应用**

## Android 设备安装方法

![Android PWA 安装](/install-pwa/android.webp)

1. 使用 Chrome 浏览器打开 Leximory
2. 点击右上角菜单按钮
3. 选择**将此网站安装为应用**
4. 按提示完成安装

## 安装后体验

安装 PWA 版本的 Leximory 后，您可以：

- 从设备主屏幕直接启动
- 获得全屏沉浸式体验
- 支持离线访问部分功能
- 接收重要通知提醒
